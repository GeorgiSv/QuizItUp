# QuizItUp

Softuni Web Project.
Application for creating and taking quizes where you can earn trophies and compete with other playes.


## Technologies used:

- ASP.NET Core
- Entity Framework Core
- SQL Server
- JavaScript
- jQuery
- Cloudinary
